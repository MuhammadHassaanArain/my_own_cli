**My Own CLI** is a terminal-based AI assistant built using Python, `agent_sdk`, and `rich`.  
It allows you to interact with an AI agent directly from your command line in a conversational way.

---

## Features

- Interactive terminal assistant
- Colorful and user-friendly CLI interface using `rich`
- Async support for smooth performance
- Goal-driven AI responses via `agent_sdk`

---

## Requirements

- Python 3.11+  
- `agent_sdk` (your custom agent SDK)  
- `rich` Python library  

Install dependencies with:

```bash

git clone https://github.com/yourusername/my_own_cli.git
cd my_own_cli

pip install -r requirements.txt

# Run CLI
python -m cli.main
```
